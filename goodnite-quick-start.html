Быстрый запуск:

  1. Распакуйте коробку с подушкой
  2. Подключите источник питания с переходным кабелем к электронному блоку управления
  3. Положите подушку goodnite на кровать плотной стороной вниз (не кладите ещё одну подушку поверх goodnite)
  4. Блок управления можно положить на пол
  5. Включите питание на блоке управления
  6. Подушка работает. Приложение Nitelink2 для работы подушки необязательно (но даст много информации)
goodnite готов помочь вам снизить храп!

Хотите посмотреть демонстрационный режим?

  • Выключиет кнопку питания на блоке управления (если она включена)
  • Нажмите и держите нажатой кнопку Bluetooth на блоке управления
  • Включите кнопку питания не отпуская кнопку Bluetooth
  • Загорится синий огонек Bluetooth. Отпустите кнопки
  • Для демонстрации работы подушка будет надувать и сдувать разные камеры
  • Выключите питание на блоке управления для окончания демонстрации работы
